# SmartChef AI 🍳

SmartChef AI is an AI-powered cooking assistant that helps users create recipes using ingredients available at home.

The system combines multiple AI technologies:

* Computer Vision for ingredient recognition using [Teachable Machine](https://teachablemachine.withgoogle.com?utm_source=chatgpt.com)
* Recipe generation using [ChatGPT](https://chat.openai.com?utm_source=chatgpt.com) 
* AI-generated food images using [Google Gemini](https://gemini.google.com?utm_source=chatgpt.com).
* Text-to-Speech narration using [TTSMaker](https://ttsmaker.com?utm_source=chatgpt.com)

## Workflow
Ingredient Image
      ↓
AI detects ingredients
      ↓
AI generates recipe
      ↓
AI creates dish image
      ↓
Recipe converted to speech

#Features

* Ingredient detection
* Automatic recipe generation
* AI food image creation
* Voice cooking instructions

This project demonstrates how multiple AI technologies can work together to create a smart and interactive cooking assistant.
